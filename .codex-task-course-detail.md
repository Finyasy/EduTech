# Task: Course detail page (course overview + lesson list)

## Goal
Add a **Course detail page** at `src/app/(learning)/courses/[courseId]/page.tsx` that shows:
1. **Course overview**: title, description, grade level, and lesson count
2. **Lesson list**: ordered list of lessons, each linking to `/courses/[courseId]/lessons/[lessonId]`

## Requirements
- **Route**: Create `src/app/(learning)/courses/[courseId]/page.tsx` (dynamic segment `[courseId]`).
- **Data**: Use `getCourse(courseId)` and `listLessons(courseId)` from `@/lib/server/data`. Call both in the server component; if course is null, call `notFound()`.
- **Layout**: Use `SiteHeader` and the same wrapper/layout as the courses list and lesson pages (e.g. `min-h-screen bg-amber-50`, `mx-auto max-w-6xl px-6 pb-16 pt-10`).
- **Styling**: Match existing app style (amber/slate, rounded-3xl cards, `var(--font-display)` for headings, orange accent, text-xs uppercase tracking for labels).
- **Course overview block**: Show course title (main heading), description, grade level (small label), and lesson count.
- **Lesson list**: Ordered list (by lesson order). Each item: lesson title + link "Start" or "View" to `/courses/[courseId]/lessons/[lessonId]`. First lesson can have a prominent "Start course" CTA if you like.
- **Links**: From the courses list page, "View course" or similar should go to `/courses/[courseId]` (this new page). Optionally add a "View course" link on course cards in `src/app/(learning)/courses/page.tsx` that points to `/courses/[courseId]` so users can open the detail page before starting.

## Reference files
- Course list: `src/app/(learning)/courses/page.tsx`
- Lesson page: `src/app/(learning)/courses/[courseId]/lessons/[lessonId]/page.tsx`
- Data API: `src/lib/server/data.ts` (`getCourse`, `listLessons`, types `CourseOverview`, `LessonDetail`)

Do not change the lesson page or the API; only add the new course detail page and optionally add a "View course" link on the courses list.
